### Installation

<CodeGroup>

```bash npm
npm install @threadify/sdk
```

```bash yarn
yarn add @threadify/sdk
```

```bash pnpm
pnpm add @threadify/sdk
```

</CodeGroup>

## Import

<CodeGroup>

```javascript ES Modules (recommended)
import { Threadify } from '@threadify/sdk';
```

```javascript CommonJS
const { Threadify } = require('@threadify/sdk');
```

</CodeGroup>
