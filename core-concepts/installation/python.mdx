### Installation

<CodeGroup>

```bash pip
pip install threadify-sdk
```

```bash uv
uv pip install threadify-sdk
```

</CodeGroup>

## Import

<CodeGroup>

```python Standard Import
from threadify import Threadify
```

```python With Alias
import threadify
```

</CodeGroup>

## Requirements

- **Python version**: 3.9 or higher
- **Dependencies**: Automatically managed by pip
